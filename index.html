Projects

Blue Eating

Hack PSU - March 2024

Blue Eating is a project created by myself and three other very talented software developers for a coding competition. The theme of the competition was Earth Day, therefore our website was based around the most famous 'Blue Zones.' It features a login page for users to create an account, as well as a place to upload their own healthy recipes from Blue Zones, and the ability to search recipes on the site based on dietary restrictions.

Challenges

  1. Staying up and finishing a site in 24 hours
  2. Learning React.js in 24 hours only based off knowledge of Angular
  3. Working as a team under a time constraint and staying organized on our individual tasks
  4. Presenting to a multitude of different judges and explaining exactly how our code works

Global Talk

Personal Project - February to May 2024

Global Talk was a personal project done for school that I extended in my own time. It is an app done in Java using Android Studio, that helps users quickly see how to ask for directions, proper greetings based on formality, and how to order. Users can select from six different language packs, each language having three options for vocabulary/phrases, and the ability to share the app on social media platforms.

Challenges

  1. Coming up with an eyecatching but easy to view design
  2. Using my old knowledge of Java to build an app
  3. Research and documentation on best app design practices
